# 🐉 La Libélula de Gael 💙

Página web tierna y **kawaii** sobre la libélula, hecha para complementar la
exposición escolar de **Gael (3 años)**.

La página es de una sola pantalla, pensada para niños:

- 🔎 Las **6 cosas geniales** de la libélula (ojos grandes, cuatro alas, vuela rápido, come insectos, vive en el agua).
- 🌟 Curiosidades fáciles (¿Sabías que...?).
- 🎨 Galería de libélulas de colores.
- 📸 Más fotos lindas.
- 🎮 Mini-juego con confeti.
- 🥰 Mascota libélula kawaii con carita feliz.

## 🚀 Ver la página

Se publica con **GitHub Pages**: `https://unimauro.github.io/libelula-gael/`

## 🧩 Cómo está hecha

- `index.html` — todo en un solo archivo (HTML + CSS + JS, sin dependencias).
- `img/` — las fotos de las libélulas.
- Fuentes tiernas *Baloo 2* y *Fredoka*.
- Funciona en celular y computadora 📱💻.

Hecho con mucho cariño para Gael y su exposición. 💖
